# Codex Adapter

The canonical Codex-compatible skill folder is:

```txt
skill/hoi4-modding/
```

Install that directory with Codex's skill installer when this repository is published.

Example shape:

```txt
$skill-installer install https://github.com/postigodev/hoi4-agent-skill/tree/main/skill/hoi4-modding
```

After installing, restart Codex so the skill metadata is loaded.

Do not copy adapter text into the canonical skill. The source of truth is `skill/hoi4-modding/SKILL.md`.
